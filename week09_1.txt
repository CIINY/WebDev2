1.브라우저 객체 모델
브라우저의 기능들을 제공하는 객체들(이해하고 정리해놓을 필요는 있다.)
->BOM (Browser Object Model) / (screen까지 다 객체 이름)
	-> window  (속성들에 대한 객체임)
	: 브라우저 객체 구조에서 가장 최상위의 객체
		- 브라우저의 기본 토대를 제공하는 중요한 객체
		- 윈도우 생성, 수정, 삭제 등을 포함하는 모든 동작의 출발점
		- var로 선언한 변수들은 모두 window객체의 속성이 됨
	 	- 공개된 표준이 적용되는 객체가 아니지만 브라우저마다 모두 제공함
	 메소드 - alert(msg) : 경고창으로 메시지 출력 (msg : 문자열)
	         - prompt(msg, <default msg>) : msg를 출력하고 입력 문자열 입력바디
	         - comfim(msg) : msg를 출력하고 [확인], [취소] 선택  (comfim? comfirm?)
			let name = prompt("이름을 입력하세요", '홍길동');     -> default값인 '홍길동'에서도 취소를 누르면 값이 자동으로 날라감. (입력한 이름은 null 입니다.)
          			alert("입력한 이름은 " + name + " 입니다.");

			let result = confirm("성인입니까?");
          			alert(`result = ${result}`);  	-> 취소 : result = false

			let result = confirm("성인입니까?");
